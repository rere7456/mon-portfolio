# PROJET_MASTER : Portfolio Photographe (Inspiration Stéphane Basset)

## Objectif
Créer un site portfolio élégant et responsive pour un photographe.
*Identité visuelle : Minimalisme, noir et blanc avec accents colorés, typographie épurée.*

## Structure du Projet
- **index.html** : Structure principale, intégration Tailwind.
- **style.css** : Configuration Tailwind (`bg-slate-950`, `text-slate-200`, `accent-emerald-500`).
- **script.js** : Logique de filtrage avec animations `transition-all` et `duration-300`.
- **README.md** : Documentation du projet.

## Étapes de développement
1. **Scaffolding** : Mise en place avec Tailwind CSS.
2. **Design** : Palette : `bg-slate-950` (fond), `text-slate-100` (titres), `text-slate-400` (textes). Accents : `emerald-500`.
3. **Galerie** : Grille `grid-cols-1 md:grid-cols-2 lg:grid-cols-3` avec `hover:scale-105` et `transition-transform`.
4. **Interactivité** : Smooth scroll (`scroll-smooth`), animations d'apparition `opacity-0 animate-fade-in`.
5. **Finalisation** : Optimisation responsive et performances.
