# Plan du Projet : Portfolio Renan Basset

## 1. Vision & Style
- Style : Artistique, organique, animations fluides (similaire à davidmilan.com).
- Technologies : Next.js, Tailwind CSS, Framer Motion, Supabase.

## 2. Structure Technique (Fichiers vides à créer)

- `src/app/admin/page.tsx`
- `src/components/ProjectCard.tsx`
- `src/components/OrganicAnimation.tsx`

## 3. Étapes de Développement
1. **Architecture** : Next.js + Tailwind + Framer Motion.
2. **Backend** : Supabase Setup (Auth, DB pour projets, Storage pour images).
3. **Frontend Public** : Landing Page, Galerie de projets avec animations.
4. **Admin** : Interface sécurisée pour CRUD de projets.
