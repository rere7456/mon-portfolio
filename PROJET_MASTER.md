# Plan du Projet : Portfolio Renan Basset

## 1. Vision & Style
- Style : Artistique, organique, animations fluides et immersives (inspiré par davidmilan.com, avec une touche de poésie visuelle).
- Palette de couleurs : Sombre et organique (ex: verts profonds, bruns terreux, gris ardoise) avec des touches de couleurs vives pour les accents (ex: corail, or).
- Animations : Utilisation avancée de Framer Motion pour des transitions douces, des effets de parallaxe sur le scrolling, et des micro-interactions réactives.
- Technologies : Next.js, Tailwind CSS, Framer Motion, Supabase.

## 2. Structure Technique (Fichiers vides à créer)
- `src/app/page.tsx` (Public Portfolio Landing)
- `src/app/admin/page.tsx` (Admin Dashboard)
- `src/components/ProjectCard.tsx`
- `src/components/OrganicAnimation.tsx`
- `src/lib/supabase.ts`

## 3. Étapes de Développement
1. **Architecture** : Next.js + Tailwind + Framer Motion.
2. **Backend** : Supabase Setup (Auth, DB pour projets, Storage pour images).
3. **Frontend Public** : Landing Page, Galerie de projets avec animations.
4. **Admin** : Interface sécurisée pour CRUD (Create, Read, Update, Delete) de projets.

## 4. Stratégie Visuelle (Designer)
- Palette : Sombre et organique (ex: verts profonds, bruns terreux, gris ardoise) avec des touches de couleurs vives pour les accents (ex: corail, or).
- Couleurs Tailwind : Utiliser une configuration sombre avec une palette personnalisée.
- Animations : Micro-interactions fluides et organiques (voir Framer Motion), scrolling parallax, transitions douces entre les sections.
